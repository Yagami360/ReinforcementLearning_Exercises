強化学習の学習環境用の迷路探索問題。<br>
等確率で表現された４つの移動候補（上下左右）から１つの方向を無作為に選択し、これを繰り返すことで、最終的に目的地に到着させる。<br>
Unity ML-Agents のフレームワーク（`Academy`,`Brain`,`Agent`クラス など）を参考にして実装している。<br>
